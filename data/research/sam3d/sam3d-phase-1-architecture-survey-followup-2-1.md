### 2. SAM 2 — **VERIFIED REAL**

**Confidence: 0.99**

SAM 2 (Ravi et al., 2024) is real:
- **Paper**: "SAM 2: Segment Anything in Images and Videos"
- **ArXiv**: 2408.00714 (August 2024)
- **Code**: github.com/facebookresearch/sam2
- **Key advance**: Extends SAM to video with a streaming memory architecture

**Architecture**:
- Image encoder: Hiera (hierarchical ViT) — more efficient than SAM's ViT-H
- Memory encoder + memory bank: stores past frame features + masks
- Memory attention: cross-attends current frame to memory bank
- Mask decoder: same prompt-based design as SAM 1

**Memory mechanism**:
$$\mathbf{M}_t = \text{concat}[\mathbf{m}_{t-1}, \mathbf{m}_{t-2}, ..., \mathbf{m}_{t-k}]$$
$$\hat{\mathbf{f}}_t = \text{CrossAttn}(\mathbf{f}_t, \mathbf{M}_t)$$

where $\mathbf{f}_t$ is the current frame's features and $\mathbf{M}_t$ is the memory bank (up to $k=6$ recent frames + up to 16 prompted frames).

**Benchmarks**: SA-V dataset (50.9K videos), achieves SOTA on video object segmentation. Real numbers, real code.

$$(\hat{\theta}, \hat{\beta}, \hat{\pi}) = f_\phi(\text{crop}(I, \text{bbox}))$$

where $f_\phi$ is a ViT-based regressor and $\pi$ are weak-perspective camera parameters. The output is in a **camera-relative coordinate frame**, not world coordinates.

**WHAM** adds world grounding via:

$$\mathbf{R}^w_t, \mathbf{t}^w_t = g_\psi(\hat{\theta}_{1:T}, \mathbf{a}_{1:T})$$

$$\hat{\mathbf{t}}^w_t = \hat{\mathbf{t}}^w_{t-1} + \Delta\hat{\mathbf{t}}_t$$

where $\Delta\hat{\mathbf{t}}_t$ is predicted from the motion decoder, **not from depth estimation**. WHAM sidesteps the depth problem entirely by learning motion priors from mocap data.
